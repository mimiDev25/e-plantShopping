# coding-project-template
This website, e-plantShopping, is a plant nursery website. It consists of a home/about page, a product page, and a shopping cart display page. The file structure is simple. The redux store contains a single slice for cart items. There are three function components within the App.jsx parent. AboutUs.jsx is a stateless component that renders paragraphs within the App.jsx component's "About" page. ProductList.jsx contains the array of plant data (category, name, price, etc.) and renders this data in cards via the map method. Lastly, CartItem.jsx accesses the CartSlice via the redux store, rendering the selected plants in cards and allowing updates to selected items and their quantities. This is e-plantShopping.
